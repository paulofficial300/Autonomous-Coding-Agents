This script isn't just a basic chatbot—it is an autonomous ReAct (Reason + Act) agent powered by Gemini (accessed via the OpenAI SDK compatibility layer). It can think through problems, write files, execute system commands, and search the web, all while keeping you in the driver's seat for potentially destructive actions.What the Agent DoesWhen you give the agent a complex, multi-step task (e.g., "Research the latest Gemini models and save a summary to a text file"), it doesn't just guess the answer. It enters a structured loop:Thought: It analyzes your request and decides on the next logical step.Action: It selects a tool (like web_search, write_file, or execute_bash) and outputs a JSON command.Observation: The Python script parses that JSON, executes the tool locally, and feeds the results back to the agent.Resolution: The loop repeats until the agent has gathered enough context to formulate a final answer.Built-in Safety (Human-in-the-Loop)Because this agent has access to your local operating system, it includes a strict safety catch. Safe actions (like searching the web or reading files) happen automatically. However, if the agent attempts to modify your system—such as writing a file or executing a terminal command—the script pauses and asks for your permission (e.g., Agent wants to run: python script.py. Allow? (y/n)). If you decline, the agent is informed of the block and must find an alternative solution.ResilienceThe script uses the tenacity library to handle network hiccups or API rate limits. If a request fails, it automatically applies exponential backoff, waiting a few seconds before retrying, ensuring the loop doesn't crash halfway through a complex task.How to Set Up and Use the Code1.Install dependencies:The script requires the OpenAI SDK, Tenacity for retries, and dotenv for managing environment variables. Open your terminal and run:Bashpip install openai tenacity python-dotenv
2.Configure your API key:Get a Gemini API key from Google AI Studio. The script looks for an environment variable named GEMINI_API_KEY. You can set this in your terminal:Mac/Linux:Bashexport GEMINI_API_KEY="your_api_key_here"
Windows (PowerShell):PowerShell$env:GEMINI_API_KEY="your_api_key_here"
(Alternatively, create a .env file in the same folder as your script and add GEMINI_API_KEY="your_api_key_here" inside it.)3.Save the script:Save the provided Python code into a new file named agent.py. Ensure you are working in a directory where you don't mind the agent creating or reading files.4.Run and interact:Execute the script from your terminal:Bashpython agent.py
You will see a You: prompt. The agent is now waiting for your command.
